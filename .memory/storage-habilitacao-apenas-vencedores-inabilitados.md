# `storage/habilitacao-apenas-vencedores-inabilitados`
```
Categoria "Habilitação" em gestão de storage deve exibir APENAS fornecedores VENCEDORES e INABILITADOS/REJEITADOS, não todos os participantes do processo. Para Compra Direta: buscar planilha consolidada mais recente, identificar fornecedores com `eh_vencedor: true` em pelo menos um item, adicionar estes vencedores + fornecedores explicitamente rejeitados. Para Seleção de Fornecedores: mesma lógica - apenas vencedores e inabilitados. Esta regra é crítica porque a planilha de habilitação destaca em verde (vencedores) e vermelho (inabilitados) - portanto apenas estes devem aparecer na categoria Habilitação, não todos que participaram.
```
